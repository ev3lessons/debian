# EV3Lessons Debian Apt server for BrickPi
Distributions:<br>
  jessie
  
Architectures:<br>
  armhf,<br>
  armel,<br>
  all

Packages:<br>
  brickpi-python,<br>
  brickpi-boot,<br>
  avrdude
  
Add to system:<br>
 sudo nano /etc/apt/sources.list <br>
 add "deb http://ev3lessons.com/debian jessie main" to the bottom of the file
 Ctrl-X --> Y --> Enter
